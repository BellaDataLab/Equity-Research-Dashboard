# Equity Research Dashboard

A full-stack financial analytics platform that transforms market, fundamental, and analyst data into an interactive equity research experience.

![Dashboard](dashboard.png)

## Overview

This dashboard enables investors and analysts to evaluate public companies through:

- Real-time market performance
- Fundamental valuation metrics
- Financial statement analysis
- Analyst consensus and price targets
- Historical price visualization

## Tech Stack

Python · Flask · Pandas · yFinance · JavaScript · Chart.js

- Financial Analytics
- Data Engineering
- API Development
- Dashboard Design
- Data Visualization
- Full-Stack Development
